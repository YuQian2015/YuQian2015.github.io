快速创建Linux用户

连接Linux服务器

执行创建用户命令

假设需要创建的用户为 tom, 在终端执行： useradd -m 用户名 创建一个 admin 用户和 home 目录

查看用户

执行 compgen -u 查看用户

为用户设置密码

使用 passwd 用户名 为用户设置密码

键盘输入密码两次

添加用户到root组

使用 usermod -a -G root 用户名 添加用户到root组， 其中 -a 表示添加，-G 表示添加到组

修改bash外壳

使用命令 chsh -s /bin/bash 用户名
